# ProposalAgent


*Proposal Agent ** is an AI engineering assistant that leverages LLMs and our internal APIs to automate and accelerate downloading of proposals.

## 📖 About

This project enables you to:

- Run LLMs like  `mistral`  **locally** via Ollama.
- Use a clean and modular Python backend.
- Format your code consistently using `black`.
- Work collaboratively using GitLab pull request workflow.

> **Requirements**

> - Python 3.11.0
> - [Ollama](https://ollama.com/) installed locally
> - `git` and basic terminal usage

### Installation

This setup uses **Python 3.11** and installs all requirements via `pip` , offering a reproducible and isolated environment.


```bash
pip install virtualenv
python3.11 -m venv myenv
source myenv/bin/
pip install -r requirements.txt
pip install black


```

### Code Formatting

To format the updated files:

```bash

black filename.py

```
